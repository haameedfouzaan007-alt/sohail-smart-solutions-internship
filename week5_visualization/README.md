# Week 5 — Visualization with matplotlib

## Task Overview

In this task, I focused on learning how to visualize data using the matplotlib library in Python. After cleaning a messy student dataset in the previous task, I used that cleaned data to create different types of charts.

The purpose of this task was not just to create charts, but also to understand what the data is telling us in a more visual and meaningful way. Instead of only looking at rows and columns in a table, I was able to see patterns, trends, and comparisons more clearly through graphs.

This task helped me realize how important visualization is when working with data, especially when trying to explain findings to others.

## Files Included

* `week5_visualization.ipynb` — Jupyter Notebook containing all the code and charts
* `cleaned_students.csv` — The cleaned dataset used for visualization
* `README.md` — This file explaining the task

## Dataset Used

For this task, I used the cleaned student dataset that I prepared in the Week 5 data cleaning task.

The dataset contains the following information:

* Student ID — A unique identifier for each student
* Name — The student's name
* University — The university the student attends
* Course — The course the student is enrolled in
* Grade — The student's grade

Since the dataset was already cleaned, I was able to directly focus on analyzing and visualizing the data without worrying about missing or incorrect values.

## What I Practiced

During this task, I practiced several important skills related to data visualization:

* Loading a cleaned CSV file using pandas
* Using matplotlib to create different types of charts
* Creating a bar chart to compare values
* Creating a histogram to understand distribution
* Creating a scatter plot to observe relationships
* Adding titles to charts to make them clear
* Labeling the x-axis and y-axis properly
* Writing simple explanations (insights) for each chart

These steps helped me understand not just how to create charts, but also how to present data in a way that others can easily understand.

## Charts Created

### 1. Bar Chart — Average Grade per Course

In this chart, I calculated the average grade for each course and displayed it using a bar chart.

This helped me compare how students are performing across different courses. It made it easy to see which courses have higher or lower average grades at a glance.

### 2. Histogram — Grade Distribution

This chart shows how grades are distributed across all students.

By using a histogram, I was able to see how many students fall into different grade ranges. This helped me understand whether most students are scoring high, low, or somewhere in the middle.

### 3. Scatter Plot — Student Number vs Grade

In this chart, I plotted student numbers against their grades.

This helped me observe how grades are spread across students. While there may not be a strong pattern, it gives a visual sense of variation and whether there are any unusual values.

### 4. Bar Chart — Number of Students per Course

This chart shows how many students are enrolled in each course.

It helped me quickly compare the size of each course and see which ones have more or fewer students.

## Reflection

This task helped me understand how powerful data visualization can be. While tables are useful for showing exact values, charts make it much easier to spot patterns, trends, and differences.

I also learned that creating a chart is not enough — it is important to make it clear and readable by adding proper titles and axis labels. This ensures that anyone looking at the chart can understand it without confusion.

Overall, this task helped me connect the process of data cleaning with data visualization. It showed me how cleaned data can be transformed into meaningful insights through simple visual tools.
