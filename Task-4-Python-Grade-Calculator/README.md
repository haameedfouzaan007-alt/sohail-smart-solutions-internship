# Task 4: Python Grade Calculator

This project is a simple Python program created as part of my internship learning task. The main purpose of this project is to practice basic Python programming and rebuild my confidence in writing small working programs.

## Project Description

The Grade Calculator takes marks for three subjects as input from the user. It checks whether the entered marks are valid, calculates the total and average, and then assigns a grade based on the average mark.

- Displaying output clearly
