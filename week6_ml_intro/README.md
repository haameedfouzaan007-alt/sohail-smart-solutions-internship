# Week 6 — Machine Learning Introduction

## Task Title

What Is Machine Learning? — From Data Analysis to Prediction

## Task Summary

This task introduced me to the basic idea of machine learning.

In the previous weeks, I worked on data cleaning, visualization, and EDA using pandas and matplotlib. This task helped me understand how data analysis can move into prediction.

The main focus of this task was to understand machine learning concepts, identify features and labels, and split data into training and test sets.

## Dataset Used

The dataset used in this task is the Iris dataset from scikit-learn.

This dataset contains flower measurements for three Iris flower species:

- Setosa
- Versicolor
- Virginica

## Features and Label

The features, also called X, are:

- sepal length
- sepal width
- petal length
- petal width

The label, also called y, is:

- flower species

## Task Type

This is a classification task because the goal is to predict a category.

In this task, the category is the type of Iris flower.

## Key Learning

The main idea I learned is that machine learning uses examples to learn patterns.

Instead of writing every rule manually, we give the computer data and correct answers. The model then learns from the training data and should be tested on data it has not seen before.
