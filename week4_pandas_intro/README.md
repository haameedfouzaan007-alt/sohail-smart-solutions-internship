## OOP Classes vs pandas

In my OOP project, I used classes like `Student` and `Course` to organize the data. This was useful for building a small program and understanding object relationships.

In pandas, the data is stored in a table format called a DataFrame. This makes it easier to inspect, filter, and perform calculations on the data.

OOP is useful for designing programs and working with objects, while pandas is more useful for handling datasets and performing analysis. Since AI work depends heavily on data, pandas is an important tool for me to learn.
