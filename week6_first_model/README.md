# Week 6 — Your First Machine Learning Model

## Task Title

Your First Model — Training & Making Predictions

## Task Summary

This task introduced me to training my first machine learning model.

In the previous task, I learned the basic idea of machine learning and split the Iris dataset into training and test data.

In this task, I trained a simple K-Nearest Neighbors model using scikit-learn and used it to make predictions on test data.

## Dataset Used

The dataset used in this task is the Iris dataset from scikit-learn.

The dataset contains flower measurements for three Iris flower species:

- Setosa
- Versicolor
- Virginica

## Features and Label

The features, also called X, are:

- sepal length
- sepal width
- petal length
- petal width

The label, also called y, is:

- flower species

## Model Used

The model used in this task is K-Nearest Neighbors, also called KNN.

KNN predicts based on nearby examples from the training data.

## Work Completed

- Reviewed the scikit-learn workflow
- Learned what `.fit()` means
- Learned what `.predict()` means
- Loaded the Iris dataset
- Created X and y
- Split the dataset into training and test sets
- Created a KNN classifier
- Trained the model using `.fit()`
- Made predictions using `.predict()`
- Compared predictions with actual answers
- Changed `n_neighbors` and observed the result
- Wrote a reflection about what the model got right and wrong

## Key Learning

The main thing I learned is that machine learning follows a clear workflow.

First, we create the model. Then we train it using `.fit()`. After training, we use `.predict()` to make predictions on new data.

This task helped me understand how data preparation connects to actual prediction.
