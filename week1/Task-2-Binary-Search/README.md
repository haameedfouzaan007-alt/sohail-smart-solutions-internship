# Week 1 - Task 2: Binary Search Implementation in C++

## Concept Selected
Binary Search

## Language Used
C++

## Description
This project implements Binary Search using C++. Binary Search is used to find a target value inside a sorted array.

Array used: [10,20,30,40,50,60]

Target value: 60

The program checks the middle value first. If the target is greater than the middle value, it searches the right side. If the target is smaller, it searches the left side.

## Files Included
- binary_search.cpp
- output-screenshot.png

## Expected Output
Target found at index 5

## Learning Outcome
This task helped me understand how to convert the Binary Search concept into working C++ code using low, high, mid, and a while loop.
