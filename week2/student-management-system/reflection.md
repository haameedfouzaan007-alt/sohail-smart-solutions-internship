# Week 2 Reflection – Student Management System

## What I Learned

While working on this project, I learned how to build a simple Python application using functions, lists, dictionaries, loops, and basic error handling. One thing that stood out to me was the importance of organizing code into separate functions instead of putting everything in one place. Doing this made the program much easier to understand, test, and update.

## Which Python Concept Was Most Useful

The Python concept that helped me the most was dictionaries. They made it easy to store different details about each student, such as their ID, name, university, and major. By combining dictionaries with a list, I was able to manage multiple student records in a simple and organized way.

## What Was Difficult

The most challenging part for me was implementing the search and delete features. At first, I found it confusing to loop through a list of dictionaries and locate a specific student using their ID. It took some practice before I fully understood how the logic worked.

## How I Solved Problems

Whenever I got stuck, I tested the program step by step and used sample student records to check whether each feature was working correctly. Breaking the project into smaller functions also helped me identify and fix issues more easily. In addition, I referred to Python learning resources and experimented with small examples before applying the concepts to the final program.

## What I Want to Improve Next

Going forward, I would like to strengthen my understanding of Python data structures and improve my problem-solving skills. I am also interested in learning more about object-oriented programming and how to store data permanently using files or databases instead of keeping everything in memory while the program is running.

## Overall Reflection

Overall, this project was a good learning experience. It helped me become more comfortable with Python programming and gave me practical experience working with data structures and functions. It also improved my confidence in using GitHub for project organization and documentation. Completing the project showed me how important planning, testing, and debugging are when developing software.
