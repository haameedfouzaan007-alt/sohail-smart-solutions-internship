# Student Management System

## Project Overview

This project was developed as part of my Week 2 internship tasks at Sohail Smart Solutions. The goal was to create a console-based Student Management System using Python while applying core programming concepts such as functions, lists, dictionaries, loops, input validation, and exception handling.

The system provides a simple and user-friendly menu-driven interface that allows users to manage student records efficiently.

---

## Features

The Student Management System includes the following functionalities:

* Add a Student
* View All Students
* Search for a Student by Student ID
* Delete a Student by Student ID
* Exit the Program Safely

---

## Student Information Stored

Each student record contains the following details:

* Student ID
* Student Name
* University
* Major

Student data is stored using Python dictionaries within a list structure, allowing efficient management and retrieval of records.

This project demonstrates the use of the following Python concepts:

* Functions
* Lists
* Dictionaries
* Loops
* Input Validation
* Exception Handling (`try` / `except`)
* Conditional Statements

## How to Run the Program

1. Open the project folder.
2. Run the Python file using the following command:

```bash
python student_management_system.py
```

3. Choose an option from the menu.
4. Follow the on-screen instructions to add, search, view, or delete student records.
